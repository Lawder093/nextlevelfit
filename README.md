# Next Level Fit

Tienda web demo enfocada en suplementos deportivos y rendimiento físico.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- JSON

## Características

- Diseño responsive
- Carrusel principal
- Catálogo dinámico desde JSON
- Filtrado por categorías
- Botón de compra por WhatsApp
- Estructura modular y escalable

## Vista previa

https://lawder093.github.io/nextlevelfit/

## Objetivo del proyecto

Desarrollar una landing page moderna para una marca fitness con catálogo interactivo, pensada como proyecto de portafolio y base para futura tienda real.
